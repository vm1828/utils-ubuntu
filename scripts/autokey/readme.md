# NUMPAD keys used in Autokey:

| Code         | Key    | Command      |
| ------------ | ------ | ------------ |
| keycode 180  | HOME   | Ctrl+S       |
| keycode 163  | mail   | Ctrl+C       |
| keycode 148  | calc   | Ctrl+Shift+V |
| <np_home>    | home   | Esc          |
| <np_up>      | up     | Ctrl+Z       |
| <np_page_up> | pageup | Flameshot    |

# Commands

Mappings: `xmodmap -pke`  
To change mappings: `xmodmap -e "keycode 180 = F13"`
